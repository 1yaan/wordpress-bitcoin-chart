WP Bitcoin Chart Plugin

WP Bitcoin Chart Pluginは、Wordpressのプラグインです。
プラグインの設定はありません！
ショートコードを発行するだけで、すぐにBTC/JPYのグラフをあなたのWordpressに挿入することができます。

WP Bitcoin Chartプラグインで使っているデータは、Cryptwatch BTC/JPYを使用しています。

WP Bitcoin Chartプラグインでは、Chartjsを使用してグラフを表示します。

WP Bitcoin Chartプラグインでは、5min, 30min, 60min, 1dayのデータを表示できます。表示期間の初期値は30日です。

This plugin can be easily installed without knowing PHP!


This plugin will enter all the settings in the shortcode. Look at this page and look for your favorite setting!

ショートコードについて

このプラグインには3つのショートコードが設定されています。

  • グラフを表示するショートコード [wp_bitcoin_chart_view]
  • 最終取引価格を表示するショートコード [wp_bitcoin_chart_transaction_price]
  • 24時間の相場変動を表示するショートコード [wp_bitcoin_chart_market_price]

グラフを表示するショートコード 例

グラフを表示するショートコードのみ、パラメーターを設定することができます。パラメーターによって、いろいろな種類のグラフを表示することができます.

[wp_bitcoin_chart_view name="WPBITCHART2" op=1 hp=1 op_color="PINK" hp_color="PURPLE" tool_position="none"]

Themeファイル内でショートコードを利用する場合 例

WordPressでは、テーマの中にプラグインのショートコードを設置する関数が準備されています。このdo_shortcodeをテーマファイル内に設置することで、ショートコードを利用することができます.

<?php echo do_shortcode('[wp_bitcoin_chart_view name="WPBITCHART2" op=1 hp=1 op_color="PINK" hp_color="PURPLE" tool_position="none"]'); ?>

ショートコードで使用できるパラメーター

ショートコードで以下のすべてのパラメーターを使用できます。

オプション 概要 初期値 入力できる値 備考
name このショートコードの名前 WPBITCHART 半角英数字（小文字/大文字） それぞれのグラフで、UNIQUEになるように設定してください。
market 市場 bitflyer bitflyer, gdax, bitmex等が指定可能です。 参照先のcryptowatch.jpをご覧ください。
exchange 為替の種類 btcjpy btcjpy, btcusd等が指定可能です。 参照先のcryptowatch.jpをご覧ください。
periods データの間隔 86400 300, 1800, 3600, 86400の4種類を指定可能です。 300秒=5分、1800秒=30分、3600秒=1時間、86400秒=1日です。
op_color 開始金額のグラフの色 Red CSSで利用可能なカラーネームを指定可能です。 参考
英名："Red", 16進表記："#FF0000"
hp_color 最高金額のグラフの色 Green CSSで利用可能なカラーネームを指定可能です。 参考
英名："Red", 16進表記："#FF0000"
lp_color 最低金額のグラフの色 Blue CSSで利用可能なカラーネームを指定可能です。 参考
英名："Red", 16進表記："#FF0000"
cp_color 終了金額のグラフの色 Yellow CSSで利用可能なカラーネームを指定可能です。 参考
英名："Red", 16進表記："#FF0000"
vo_color 取引量のグラフの色 Magenta CSSで利用可能なカラーネームを指定可能です。 参考
英名："Red", 16進表記："#FF0000"
op 開始金額のグラフを表示するかどうか 0 0=表示しない, 1=表示する
hp 最大金額のグラフを表示するかどうか 0 0=表示しない, 1=表示する
lp 最低金額のグラフを表示するかどうか 0 0=表示しない, 1=表示する
cp 終了金額のグラフを表示するかどうか 0 0=表示しない, 1=表示する
vo 取引量のグラフを表示するかどうか 0 0=表示しない, 1=表示する
from グラフをいつから表示するか 今日から1ヶ月前 "Y-m-d H:i"形式で入力してください。
to グラフをいつまで表示するか 今日まで "Y-m-d H:i"形式で入力してください。
tool_position グラフ切替ツールをどこに表示するか top top=グラフの上
bottom=グラフの下
both=グラフの上下両方
none=表示しない
