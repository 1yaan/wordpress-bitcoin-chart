== Description ==

This plugin displays the data of cryptowatch.jp using Chartjs.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-bitcoin-chart` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the 'WP Bitcoin Chart' screen to configure the plugin.
